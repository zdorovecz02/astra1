Что уже сделано в первом семестре

1. Проведен примерный анализ объекта программирования.

Принято решение: для разработки ПО для Пущинской радиоастрономической обсерватории использовать в качестве языка программирования PYTHON

Почему PYTHON?

С одной стороны, PYTHON имеет сравнительно невысокую скорость выполнения программ. Однако, этот недостаток компенсируется уменьшением времени разработки программы. В среднем программа на PYTHON в 2-4 раза компактнее, чем ее аналог на С++ или Java. В нашем случае PYTHON подходит лучше всех других языков программирования. Поэтому, начато изучение базы языка PYTHON.

2. Определена команда и роль каждого в работе над проектом.

На данном этапе разработки сайта «Отчет по проекту» созданы 2 команды:

1-я команда «Инженеры» (2 человека + капитан)

2-я команда «Дизайнеры» (2 человека + зам. капитана)

В процессе работы команды или члены команд могут меняться ролями. Будет зависеть от постановки задач на определенный временной промежуток, личных качеств и компетенций членов команды. Это позволит опытным путем определить предрасположенность к той или иной роли каждого из участников.

3. Создан Веб сайт для удобства представления информации по ходу работы над проектом.

Для этого установили среду разработки создания приложений WebStorm. Приступили к изучению открытой платформы для разработки веб-приложений Angular.

Почему Angular?

Предоставляет такую функциональность, как двустороннее связывание, позволяющее динамически изменять данные в одном месте интерфейса при изменении данных модели в другом, шаблоны, маршрутизация и так далее. Одной из ключевых особенностей Angular является то, что он использует в качестве языка программирования TypeScript. Поэтому, начали знакомиться с основами работы языка программирования TypeScript.

4. Начали разрабатывать дизайн сайта, используя HTML — язык гипертекстовой разметки и CSS — язык описания стилей.
